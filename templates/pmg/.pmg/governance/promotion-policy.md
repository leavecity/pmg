# Promotion Policy

A memory item can be promoted to confirmed memory only when at least one condition is true:

- explicitly approved by the user or maintainer
- repeatedly observed and not contradicted
- supported by existing project documents
- derived from an accepted ADR

Promotion should record:

- source
- reason
- date
- reviewer or approving authority
- files changed

Do not promote speculative, one-off, or task-local details.
