# Memory Lifecycle

Memory moves through these stages:

```text
Observation -> Extraction -> Classification -> Pending -> Review -> Approval -> Promotion -> Use -> Revision -> Archive
```

## Stage notes

- Observation: something noticed during work.
- Extraction: a candidate durable lesson is separated from task noise.
- Classification: the candidate is assigned a domain and confidence level.
- Pending: the candidate waits for evidence or approval.
- Review: a human or governance module evaluates the candidate.
- Approval: policy allows promotion.
- Promotion: durable memory is updated.
- Use: agents rely on the memory.
- Revision: memory is corrected as the project changes.
- Archive: memory is retired.
