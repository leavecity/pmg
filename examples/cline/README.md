# Cline Integration

PMG can complement memory-bank-style workflows by adding confidence levels, promotion policy, conflict handling, and context assembly.

Recommended workflow:

1. Keep Cline memory bank content separate from confirmed PMG memory unless promoted.
2. Use `.pmg/memory/pending.md` for candidates.
3. Use `pmg context build --task "<task>"` before implementation.
4. Promote durable knowledge only when PMG policy allows it.
