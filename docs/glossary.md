# Glossary

## Agent

An AI coding assistant or automation system that can read, modify, or reason about a repository.

## Context bundle

A bounded set of PMG files assembled for a specific task.

## Durable memory

Project knowledge expected to remain useful beyond the current task or session.

## Promotion

The act of moving pending or inferred knowledge into confirmed memory under governance rules.

## PMG

Project Memory Governance.

## Spec

A structured description of planned work, including goals, design notes, tasks, and acceptance criteria.

## ADR

Architecture Decision Record.
