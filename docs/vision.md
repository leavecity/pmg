# Vision

Project Memory Governance turns scattered project context into governed, durable, and task-addressable project knowledge.

The long-term vision is that any AI coding agent can enter a repository, read a small stable entrypoint, discover the memory governance system, assemble only relevant context, do useful work, and propose memory updates safely.

PMG should make AI-assisted development less repetitive without making repositories opaque. The system should remain inspectable by humans, reviewable in Git, and usable across vendors.
